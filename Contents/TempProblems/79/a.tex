تابع مولد نمایی برای
$a_r$
برابر است با:
$$(1 + x + \frac{x^2}{2!} + \cdots)^2(x + \frac{x^2}{2!} + \cdots)^2$$
$$= (e^x)^2(e^x-1)^2 = e^{2x}(e^{2x}-2e^x + 1)$$
$$= e^{4x} - 2e^{3x} + e^{2x}$$
$$= \sum_{r=0}^{\infty}(4^r - 2 \times 3^r + 2^r)\frac{x^r}{r!}$$
بنابراین:
$$a_r = 4^r - 2^r + 2^r$$
